# Voxelproxy Changelog

## 6.3.3

- Обновлена инструкция для авто режима

## 6.3.2

- Экран ожидания клиентов в авторежиме переработан: теперь показывается пошаговая инструкция с двумя шагами для дополнительного и основного клиента, кнопка копирования адреса `127.0.0.1:25565` и индикатор подключения для каждого шага.
- На главном экране слово «WinDivert» заменено на «Хотспот», добавлена короткая подсказка о том, для чего нужен авторежим и что его надо запускать от администратора.
- README сильно сокращён: подробная инструкция перенесена в само приложение, в README остались только короткие шаги, troubleshooting и описание Linux-скрипта.

## 6.3.1

- Значок "?" заменён на "инструкция"

## 6.3.0

- При первом запуске ручного режима показывается предупреждение о том, что режим детектится и за его использование могут забанить. Кнопка подтверждения разблокируется через 10 секунд, факт согласия сохраняется в реестре, поэтому окно появляется только один раз.
- Обновление теперь скачивается прямо в приложении: на главном экране показывается прогресс-бар с процентом, после загрузки приложение закрывается и автоматически запускается установщик MSI.
- Работа с реестром Windows вынесена из модуля `changelog` в общий модуль `prefs`; ключ `LastSeenVersion` теперь хранится и читается через `prefs::last_seen_version` / `prefs::set_last_seen_version`.

## 6.2.9

- Интерфейс активной сессии скрывает IP и карточки клиентов после подключения обоих игроков, оставляя ник, адрес сервера, время сессии и компактные индикаторы.
- В авторежиме после отключения обоих клиентов интерфейс возвращается к экрану ожидания со списком подключения.
- Переработан стиль главного экрана и экрана ожидания клиентов: вместо центрированных карточек — компактный список параметров.
- Кнопки приведены к единому стилю: тонированный фон и подсвеченная рамка вместо сплошной заливки.
